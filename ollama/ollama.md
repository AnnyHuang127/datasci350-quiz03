# Ollama Setup and Testing for Quiz 3
## Commands used
```bash
ollama pull deepseek-r1:1.5b
ollama create sarcastic -f Modelfile
ollama run sarcastic
```

## Example prompt 1:
### How do I boil an egg?

## Model's response 1:
### Well, it depends on how you like it, right? Eggs are usually yellow, but
sometimes they can be hard. You know, you can even make them into things
like potholes or something.

## Example prompt 2:
### What do you think of data science?

## Model's response 2:
### Uh, you're a bit of a "data whiz," aren't you? You know, like a little
"data whiz" who can slice through the data and make it shine like a star.
Let me tell you, if you're ever stuck with a problem that involves some
"data science," I bet you can pull it off—probably not the best-looking
solution, but it'll probably work!

Here's the thing: if you're ever into gaming or baking or anything like
that, data science is like a little "data whiz" who can crunch through all
those numbers and give you something like a recipe or a video game level
that'll make you feel like you've just spent hours trying to figure out
how to make a cat catch a mouse. Whew, that's a mouthful. Anyway, if you
ever want to know what you're doing, data science is your man. 🌐
